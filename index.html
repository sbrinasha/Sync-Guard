Featured Project: SyncGuard

AI-Driven Smart Home Automation System for Adaptive Safety

Traditional smart home safety systems rely on fixed-threshold logic, frequently leading to "alarm fatigue" caused by false positives (e.g., a spilled drink triggering a window to close). SyncGuard solves this operational plateau.

By cross-referencing local edge telemetry with global meteorological forecasts, SyncGuard implements an Adaptive Safety framework. It intelligently distinguishes between localized, benign anomalies and genuine weather threats in real-time.

Technology Stack

Python C++ Java ESP32-S3 Arduino IDE MQTT (Mosquitto) Scikit-learn (Random Forest) Flask SQLite OpenWeather API

Technology Architecture

📟

1. Edge Layer

An ESP32-S3 microcontroller gathers micro-climate data using a BME280 sensor and analog rain sensors.

📡

2. Communication

Telemetry is serialized into JSON and streamed via a lightweight MQTT broker for low-latency, bidirectional data flow.

🧠

3. AI Engine

A Python backend fetches macro-climate forecasts. A trained Random Forest Classifier fuses the data streams to verify threats.

💻

4. Presentation

A Flask-based web dashboard provides real-time visualization, AI state indication, and manual override controls.
